i 태그는 주의 텍스트와 다른 의미의 텍스트로 구분할 때 설정합니다.

by @webs 2022. 12. 28.

<i>

<i> 태그는 주의 텍스트와 다른 의미의 텍스트로 구분할 때 설정합니다.

<i> 태그는 이탤릭체를 표현하기 위한 속성이였으나, 현재는 주의 텍스트와 다른 의미의 텍스트로 구분할 때 사용하는 속성으로 변경되었습니다. <i> 태그는 기술 용어, 외국어, 생각 등을 표현할 때 설정하고, 이탤릭체로 표현됩니다. 이탤릭체로 표현하기 위해서 <i> 태그는 적절하지 않으며, CSS의 font-style: italic을 사용합니다.

정의(Definition)

  • <i> 태그는 주의 텍스트와 다른 의미의 텍스트로 구분할 때 설정합니다.
  • <i> 태그는 기술 용어, 외국어, 생각 등을 표현할 때 설정합니다.
  • <i> 태그는 이탤릭체로 표현됩니다. 이탤릭체를 표현하기 위함이라면 CSS font-style: italic을 사용합니다.
  • HTML4에서 <i> 태그는 이탤릭체를 위한 속성이였지만, HTML5에서는 주의 텍스트와 다른 의미의 텍스트로 구분할 때 사용되는 의미로 변경되었습니다.

텍스트와 관련된 태그

  • <b> 태그는 중요성을 전달하지 않고, 주의를 끌기 위한 용도로 사용할 때 설정합니다.
  • <em> 태그는 텍스트를 강조할 때 설정합니다.
  • <strong> 태그는 내용에 대한 중요성, 심각성, 긴급성을 표현할 때 설정합니다.
  • <mark> 태그는 중요성과 강조를 고려하지 않는 단순 참고용 표시를 할 때 설정합니다.
  • <i> 태그는 주의 텍스트와 다른 의미의 텍스트로 구분할 때 설정합니다.
