margin

margin 속성은 요소의 바같쪽 여백을 설정합니다.

특징 설명
기본값 margin-top : 0;
margin-right : 0;
margin-bottom : 0;
margin-left : 0;
적용 box
버전 CSS1
사용성 ★★★★★

정의(Definition)

  • margin 속성은 요소의 바같쪽 여백을 설정합니다.

비교(Compare)

  • margin 속성은 요소의 바같쪽 여백을 설정합니다.
  • margin-top 속성은 요소의 위쪽 바깥쪽 여백을 설정합니다.
  • margin-right 속성은 요소의 오른쪽 바깥쪽을 설정합니다.
  • margin-bottom 속성은 요소의 아래쪽 바깥 여백을 설정합니다.
  • margin-left 속성은 요소의 왼쪽 바깥 여백을 설정합니다.

문법(Syntax)

margin : length | % | auto

/* 기본 속성 */ 
margin: 10px;       /* CSS 단위 설정 */
margin: 1em;        /* CSS 단위 설정 */
margin: 10%;        /* 퍼센트 설정 */
margin: auto;       /* 기본값 설정 */
margin: 0 auto;     /* 가운데 정렬 */

margin: 10px;                   /* 위쪽, 오른쪽, 아래쪽, 왼쪽 */
margin: 10px 11px;              /* 위쪽,아래쪽 | 왼쪽,오른쪽 */
margin: 10px 11px 12px;         /* 위쪽 | 왼쪽, 오른쪽 | 아래쪽 */
margin: 10px 11px 12px 13px;    /* 위쪽 | 오른쪽 | 아래쪽 | 왼쪽 */

/* 전역 속성 */
margin: inherit;   /* 상속 */
margin: initial;   /* 초기화 */
margin: revert;    /* 원래대로 돌리기 */
margin: unset;     /* 설정 해제 */

속성(Property)

속성값 값(예) 설명
length margin: 2px 요소의 바같쪽 여백을 CSS단위를 통해 설정합니다.
% margin: 2% 요소의 바같쪽 여백을 %를 통해 설정합니다.
auto margin: auto 요소의 바같쪽 여백을 자동(기본)으로 설정합니다.

박스 요소와 관련된 속성

  • margin 속성은 요소의 바같쪽 여백을 일괄적으로 설정합니다.
  • margin-top 속성은 요소의 바같쪽, 위쪽 여백을 설정합니다.
  • margin-right 속성은 요소의 바같쪽, 오른쪽 여백을 설정합니다.
  • margin-bottom 속성은 요소의 바같쪽, 아래쪽 여백을 설정합니다.
  • margin-left 속성은 요소의 바같쪽, 왼쪽 여백을 설정합니다.
  • padding 속성은 요소의 안쪽 여백을 일괄적으로 설정합니다.
  • padding-top 속성은 요소의 안쪽, 위쪽 부분 여백을 설정합니다.
  • padding-right 속성은 요소의 안쪽, 오른쪽 부분 여백을 설정합니다.
  • padding-bottom 속성은 요소의 안쪽, 아래쪽 부분 여백을 설정합니다.
  • padding-left 속성은 요소의 안쪽, 왼쪽 부분 여백을 설정합니다.
  • border 속성은 요소의 테두리 속성을 일괄적으로 설정합니다.
  • border-top 속성은 요소의 테두리 위쪽 속성을 일괄적으로 설정합니다.
  • border-right 속성은 요소의 테두리 오른쪽 속성을 일괄적으로 설정합니다.
  • border-bottom 속성은 요소의 테두리 아래쪽 속성을 일괄적으로 설정합니다.
  • border-left 속성은 요소의 테두리 왼쪽 속성을 일괄적으로 설정합니다.
  • width 속성은 요소의 가로 크키를 설정합니다.
  • max-width 속성은 요소의 최대 가로 크키를 설정합니다.
  • min-width 속성은 요소의 최소 가로 크키를 설정합니다.
  • height 속성은 요소의 세로 크키를 설정합니다.
  • max-height 속성은 요소의 최대 세로 크키를 설정합니다.
  • main-height 속성은 요소의 최소 세로 크키를 설정합니다.
  • box-sizing 속성은 요소의 가로와 세로를 계산하는 방법을 설정합니다.
