padding

padding 속성은 요소의 안쪽 여백을 일괄적으로 설정합니다.

특징 설명
기본값 padding-top : 0;
padding-right : 0;
padding-bottom : 0;
padding-left : 0;
적용 box
버전 CSS1
사용성 ★★★★★

정의(Definition)

  • padding 속성은 요소의 안쪽 여백을 일괄적으로 설정합니다.

비교(Compare)

  • padding 속성은 요소의 안쪽 여백을 일괄적으로 설정합니다.
  • padding-top 속성은 요소의 안쪽, 위쪽 부분 여백을 설정합니다.
  • padding-right 속성은 요소의 안쪽, 오른쪽 부분 여백을 설정합니다.
  • padding-bottom 속성은 요소의 안쪽, 아래쪽 부분 여백을 설정합니다.
  • padding-left 속성은 요소의 안쪽, 왼쪽 부분 여백을 설정합니다.

문법(Syntax)

padding : length | % 

/* 기본 속성 */ 
padding: 10px;       /* CSS 단위 설정 */
padding: 1em;        /* CSS 단위 설정 */
padding: 10%;        /* 퍼센트 설정 */

/* 다중 속성 */ 
padding: 10px;                   /* 위쪽, 오른쪽, 아래쪽, 왼쪽 */
padding: 10px 11px;              /* 위쪽,아래쪽 | 왼쪽,오른쪽 */
padding: 10px 11px 12px;         /* 위쪽 | 왼쪽, 오른쪽 | 아래쪽 */
padding: 10px 11px 12px 13px;    /* 위쪽 | 오른쪽 | 아래쪽 | 왼쪽 */

/* 전역 속성 */
padding: inherit;   /* 상속 */
padding: initial;   /* 초기화 */
padding: revert;    /* 원래대로 돌리기 */
padding: unset;     /* 설정 해제 */

속성(Property)

속성값 값(예) 설명
length padding: 2px 요소의 안쪽 여백을 CSS단위를 통해 설정합니다.
% padding: 2% 요소의 안쪽 여백을 %를 통해 설정합니다.

예제1(Sample)

박스 요소 (bakcground, width, height, padding)

.box {
    background-color: #a2cbfa;
    width: 200px;
    height: 200px;
    margin: 0px;
    padding: 0px;
}

  • margin-bottom 속성은 요소의 바같쪽, 아래쪽 여백을 설정합니다.
  • margin-left 속성은 요소의 바같쪽, 왼쪽 여백을 설정합니다.

  • padding-right 속성은 요소의 안쪽, 오른쪽 부분 여백을 설정합니다.
  • padding-bottom 속성은 요소의 안쪽, 아래쪽 부분 여백을 설정합니다.
  • padding-left 속성은 요소의 안쪽, 왼쪽 부분 여백을 설정합니다.
  • border 속성은 요소의 테두리 속성을 일괄적으로 설정합니다.
  • border-top 속성은 요소의 테두리 위쪽 속성을 일괄적으로 설정합니다.
  • border-right 속성은 요소의 테두리 오른쪽 속성을 일괄적으로 설정합니다.

  • main-height 속성은 요소의 최소 세로 크키를 설정합니다.
