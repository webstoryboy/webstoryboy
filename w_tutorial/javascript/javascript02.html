{
    //let x = 100;

04. 배열 : 데이터 불러오기 : 2차 배열

2차 배열은 배열 안에 배열이 들어있는 형태를 말합니다. 다음은 2차 배열에 저장된 값을 불러오는 방법입니다. 

{
    const arr = [100, 200, ["javascript", "jquary"]];

06. 배열 : 데이터 불러오기 : for()문

배열을 데이터를 반복문 for문을 통해 불러 올 수 있습니다. 

{
    const arr = [100, 200, 300, 400, 500];
    
    // console.log(arr[0]);
    // console.log(arr[1]);
    // console.log(arr[2]);
    // console.log(arr[3]);
    // console.log(arr[4],);

    //for(초기값, 조건식, 증감식)
    for(let i=0; i<arr.length; i++){    //i가 배열의 수(5)보다 작은동안 해당 명령을 반복
        console.log(arr[i]);
    }
}
결과 확인하기
100 200 300 400 500

07. 배열 : 데이터 불러오기 : 중첩 for()문

for문 안에 for문을 불러오는 형태입니다. 구구단에서 쓰이는 형태입니다. 

{
    for(let i=0; i<3; i++){   
        console.log(i, "<br>");
        for(let j=0; j<3; j++){   
            console.log(j);
        }
    }
}
