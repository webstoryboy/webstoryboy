안녕하세요!. 웹스토리보이입니다. 자바스크립트 데이터 실행하기 파트입니다. 데이터를 저장하고 불러오는 방법을 익혔다면, 이제는 데이터를 실행하는 방법을 배우겠습니다. 데이터를 실행한다는 건 함수를 이용하여 실행시켜주면 됩니다.

자바스크립트에서 데이터를 실행시키는 아주 중요하기 때문에 표현하는 방법도 여러가지가 있습니다. 중요한 만큼 활용도가 높기 때문에 사용하는 방법이 아주 다양합니다. 가장 기본적인 4가지 표현을 익히고 화살표 함수 작성법을 익히는 것이 기초 단계에서는 매우 중요합니다. 안 보고 쓸 수 있을 정도로 익혀야 나중에 쉽게 이해하고 읽을 수 있습니다. 그럼 이제부터 순서대로 하나씩 익혀갑시다.

01. 선언적 함수 : 함수를 표현하는 기본적인 문법입니다.

함수는 실행문을 표현하는 방식이며, 가장 기본적인 문법입니다.

{
    function func(){
        document.write("함수가 실행되었습니다.");
    }
    func();
}
결과 확인하기
함수가 실행되었습니다.

02. 익명 함수 : 변수안에 함수를 넣고 표현하는 방법입니다.

변수안에 함수를 넣어 표현한 방식입니다. 함수 이름이 없고 변수 이름을 사용하기 때문에 익명 함수라고 불립니다.

{
    const func = function(){
        document.write("함수가 실행되었습니다.");
    }
    func();
}
결과 확인하기
함수가 실행되었습니다.

03. 매개변수 함수 : 변수 값이 전달되는 표현 방식입니다.

함수 ( )에 인자(parameter) 값을 넣을 수 있습니다. 인자에 변수값을 넣을 수 있는데, 이렇게 전달된 변수를 매개변수라고 합니다.

{
    function func(str){
        document.write(str);
    }
    func("함수가 실행되었습니다.");
}
결과 확인하기
함수가 실행되었습니다.

04. 리턴값 함수 : 리턴값을 이용한 함수 표현 방식입니다.

리턴이라 함은 결과 또는 끝을 나타냅니다. 여기서 리턴 값은 결과 값을 의미합니다. 함수 안에 실행문이 없고 변수만 있기 때문에 변수 값을 결과 값으로 지정한 형태입니다.

{
    function func(){
        const str = "실행되었습니다.";
        return str;
    }
    document.write(func());
}
결과 확인하기
함수가 실행되었습니다.

05. 선언적 함수를 화살표 함수 형식으로 변경한 방식입니다.

function이라는 키워드를 =>로 변경한 방식의 함수 표현방법입니다.

{
    // function func( ){
    //     document.write("함수가 실행되었습니다.");
    // }
    // func( );

    func = ( ) => {
        document.write("함수가 실행되었습니다.");
    }
    func( );
}
결과 확인하기
함수가 실행되었습니다.

18. 함수 유형 : 객체생성자 함수

{
    function Func(num, name, word){
        this.num = num;
        this.name = name;
        this.word = word;

        this.result = function(){
            document.write(this.num +". "+ this.name + "가 " + this.word + "되었습니다.
");
        }
    }

    //인스턴스 생성
    const info1 = new Func(1, "함수", "실행");
    const info2 = new Func(2, "자바스크립트", "실행");
    const info3 = new Func(3, "리액트", "실행");

    info1.result();
    info2.result();
    info3.result();
}
결과 확인하기
1. 함수가 실행되었습니다.
2. 자바스크립트가 실행되었습니다.
3. 리액트가 실행되었습니다.

19. 함수 유형 : 프토토타입 함수

{
    function Func(num, name, word){
        this.num = num;
        this.name = name;
        this.word = word;
    }

    Func.prototype.result = function(){
        document.write(this.num +". "+ this.name + "가 " + this.word + "되었습니다.
");
    }

    const info1 = new Func(1, "함수", "실행");
    const info2 = new Func(2, "자바스크립트", "실행");
    const info3 = new Func(3, "리액트", "실행");

    info1.result();
    info2.result();
    info3.result();
}
결과 확인하기
1. 함수가 실행되었습니다.
2. 자바스크립트가 실행되었습니다.
3. 리액트가 실행되었습니다.
