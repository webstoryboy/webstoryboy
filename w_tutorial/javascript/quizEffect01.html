소개

안녕하세요! 오늘은 퀴즈 게임을 만들어보겠습니다. 정보처리를 공부하거나 개인적인 공부를 할 때 CBT를 이용하는 경우가 있지만, 디자인이나 UX가 맘에 들지 않어 내가 만들어 봐야겠다라는 생각을 종종했습니다. 이번 기회에 기본부터 CBT까지 만들어 보면서 자바스크립트에 기본 개념을 정확히 익히는 기회가 됐으면 좋겠습니다.

배울것들

  1. 데이터 저장하기 : 변수
  2. querySelector() : 문서에서 특정 CSS 선택자로 요소를 찾는 DOM 메서드
  3. innerText : DOM 요소의 텍스트 내용을 가져오거나 설정하는데 사용되는 속성
  4. style : 자바스크립트에서 CSS 스타일을 표현하는 방법
  5. classList : 자바스크립트에서 CSS 스타일을 추가하는 방법
  6. addEventListener : 자바스크립트에서 이벤트를 처리하는 함수
