1. 설정하기

1.1 Node.js 설치하기

웹팩을 작업하기 전에 node.js를 설치해야 합니다. https://nodejs.org/ko 사이트에서 노드를 다운받고 설치합니다.

1.2 npm 설정하기

webpack 폴더를 만들고, vscode를 열고, 터미털을 열겠습니다. npm init 또는 npm init -y를 설정하면, 파일에 대한 기본 정보를 설정할 수 있습니다. -y를 붙이면 한번에 yes를 의미합니다. 폴더에는 package.json 파일이 생깁니다. 

webstoryboy@Webstoryboyui-iMac webpack2 % npm init -y
Wrote to /Users/webstoryboy/Desktop/webpack2/package.json:

{
    "name": "webpack2",
    "version": "1.0.0",
    "description": "",
    "main": "index.js",
    "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
    },
    "keywords": [],
    "author": "",
    "license": "ISC"
}

webpack과 webpack-cli를 개발자 버전으로 설치합니다. package.json 파일에 보면 webpack과 webpack-cli가 설치된 것을 확인할 수 있습니다. 

npm install webpack webpack-cli --save-dev

1.4 webpack.config 설정하기

webpack.config.js을 루트에 만듭니다. 그러면 웹팩이 자동으로 인식하게 됩니다. src 폴더를 만들고 index.js 파일을 만듭니다. npx webpack을 터미널에서 작업하면 dist 폴더와 파일이 생기는 것을 확인할 수 있습니다. 

const path = require('path');

module.exports = {
    mode: "production",     //소스 모드를 선택할 수 있음 production / development / none
    entry: {
        index: './src/index.js'     //내가 작업한 파일
    },
    output: {
        path: path.resolve(__dirname, 'dist'),  //번들러 될 폴더
        filename: 'js/[name].js',
    },
};

1.5 webpack-dev-server 설정하기

서버를 설정하고 package.json 파일에 추가된 것을 확인할 수 있습니다.

npm install webpack-dev-server --save-dev

package.json

{
    "name": "webpack2",

    "description": "",

1.6 HTML 번들러 설정하기

npm install --save-dev html-webpack-plugin

package.json에 추가된 것을 확인 할 수 있습니다.

"devDependencies": {
    "html-webpack-plugin": "^5.5.1",
    "webpack": "^5.83.1",
    "webpack-cli": "^5.1.1",
    "webpack-dev-server": "^4.15.0"
}

webpack.config.js 다음과 같이 추가해야 합니다.

const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
    mode: "production",     
    entry: {
        index: './src/index.js',
    },
    output: {
        path: path.resolve(__dirname, 'dist'), 
        filename: 'js/[name].js',
    },
    plugins: [new HtmlWebpackPlugin({
        template: './src/index.html',   //번들 전
        filename: './index.html',       //번들 후
        hash: true,                     //해싱 설정
        showErrors: true                //에러 설정
    })],
};

package.json 파일에 다음과 같이 수정 합니다. 그리고 터미널에 npm run build 설정하면, 버들러가 작동되고, npm start를 적으면 브라우저가 열립니다. 

"scripts": {
    "build": "webpack",
    "start": "webpack serve --open"
},

1.7 CSS 설정하기

CSS 로더와 style 로더를 같이 설치합니다. package.json을 확인해 보면 추가되었습니다. 

npm install --save-dev style-loader css-loader
"devDependencies": {
    "css-loader": "^6.7.4",
    "html-webpack-plugin": "^5.5.1",
    "style-loader": "^3.3.3",
    "webpack": "^5.83.1",
    "webpack-cli": "^5.1.1",
    "webpack-dev-server": "^4.15.0"
}

webpack.cofing.js 파일에 추가합니다. 

module: {
    rules: [
        {
            test: /\.css$/i,
            use: ["style-loader", "css-loader"],
        },
    ],
},

CSS 분리하기

npm install --save-dev mini-css-extract-plugin
"devDependencies": {
    "css-loader": "^6.7.4",
    "html-webpack-plugin": "^5.5.1",
    "mini-css-extract-plugin": "^2.7.6",
    "style-loader": "^3.3.3",
    "webpack": "^5.83.1",
    "webpack-cli": "^5.1.1",
    "webpack-dev-server": "^4.15.0"
}

다음을 추가합니다.

const MiniCssExtractPlugin = require("mini-css-extract-plugin");
module: {
    rules: [
        {
            test: /\.css$/i,
            // use: ["style-loader", "css-loader"],  //style에 추가됨
            use: [MiniCssExtractPlugin.loader, "css-loader"],   
            exclude: /node_modules/,    //제외
        },
    ],
},
plugins: [
    new HtmlWebpackPlugin({
        template: './src/index.html',
        filename: './index.html',
        hash: true,
        showErrors: true
    }),
    new MiniCssExtractPlugin({
        filename: "./css/style.css",
    }),
],
