소개

안녕하세요!. 웹스토리보이입니다. 자바스크립트 데이터 실행하기 파트입니다. 데이터를 저장하고 불러오는 방법을 익혔다면, 이제는 데이터를 실행하는 방법을 배우겠습니다. 데이터를 실행한다는 건 함수를 이용하여 실행시켜주면 됩니다.

자바스크립트에서 데이터를 실행시키는 아주 중요하기 때문에 표현하는 방법도 여러가지가 있습니다. 중요한 만큼 활용도가 높기 때문에 사용하는 방법이 아주 다양합니다. 가장 기본적인 4가지 표현을 익히고 화살표 함수 작성법을 익히는 것이 기초 단계에서는 매우 중요합니다. 안 보고 쓸 수 있을 정도로 익혀야 나중에 쉽게 이해하고 읽을 수 있습니다. 그럼 이제부터 순서대로 하나씩 익혀갑시다.

인덱스

  1. 01. 함수 : 선언적 함수 : 함수를 표현하는 가장 기본적인 방법입니다.

npm 초기화

npm init

웹팩 설치

npm install -D webpack webpack-cli

output-path

npx webpack --entry ./index.js --output-path public/

webpack.config

const path = require('path');
module.exports = {
    entry: "./src/index.js",
    output: {
        path: path.resolve(__dirname, "public"),
        filename: 'main.js'
    }
};
npx webpack --config webpack.config.js / npx webpack
