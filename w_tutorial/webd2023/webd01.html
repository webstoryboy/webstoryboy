소개

안녕하세요! 웹디자인 기능사 2023버전을 만들어 달라는 의뢰가 많아서. 이번에 업그레이드를 해봤습니다. 웹디자인 기능사 레이아웃은 5가지 유형이 있으며, 반응형까지 추가되었습니다. 그렇게 어렵지는 않지만, 처음하는 분한테는 상당히 어렵다고 느낄 수 있을 것 같습니다. 하지만 자격증 공부도 하면서 사이트 만드는 기본 방법을 익힌다면 실무에서도 도움이 될거라고 생각이 듭니다.
저 또한 최대한 실무에서 많이 활용할 수 있는 최신 기법으로 코딩을 하였으며, 예전에 많이 쓰던 float방식보다는 flex를 이용한 레이아웃을 작업을 하였으며, 최신 문법을 많이 활용하였습니다. 그럼 시작해보겠습니다.

01. 기본 레이아웃

기본 레이아웃부터 작업을 시작해보겠습니다. 우선 사이트의 골격을 만들어 보겠습니다. VSCODE를 실행하고 webd2023폴더를 만들고 webd01.html파일을 만들겠습니다.
!를 치고 tab버튼을 누르면 다음과 같이 파일이 자동으로 나오게 됩니다. emmit이라는 기능을 이용하면 단축키처럼 사용할 수 있습니다. VSCODE는 자동으로 내장되어 있기 때문에 시험장에서도 VSCODE를 사용한다면 크게 문제가 없을겁니다.
lang는 ko로 변경하고 title도 변경해주었습니다. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>웹디자인 기능사 레이아웃 A-1</title>
</head>
<body>
    
</body>
</html>

기본 문서를 만들었으면 이제는 시멘틱 태그1를 이용하여 사이트 내용물의 구조를 만들어보겠습니다. body 태그 안에 전체 구조를 담고 있는 wrap과 헤더, 슬라이드, 콘텐츠, 푸터 섹션을 만들겠습니다. 이 문서의 기본 구조는 헤더 영역과, 슬라이드 영역, 콘텐츠 영역, 푸터 영역으로 구분할 수 있으며, 그런 의미를 담고 있는 섹션 태그를 이용하여 작업하였습니다. 

<body>
    <div id="wrap">
        <header id="header"></header><!-- //header -->
        <article id="slider"></article><!-- //slider -->
        <section id="contents"></section><!-- //contents -->
        <footer id="footer"></footer><!-- //footer -->
    </div>
</body>

기본 구조는 작업을 했으니, 영역을 잡기 위해서 CSS를 설정하겠습니다. CSS설정은 head태그 안에 style을 선언하고 다음과 같이 작성하겠습니다. 

<style>
    * {
        margin: 0;
        padding: 0;
    }
    #wrap {
        width: 1200px;
        height: 700px;
        background-color: #e0e0e0;
        margin: 0 auto;
    }
    #header {
        width: 100%;
        height: 100px;
        background-color: #d1d1d1;
    }
    #slider {
        width: 100%;
        height: 300px;
        background-color: #bdbdbd;
    }
    #contents {
        width: 100%;
        height: 200px;
        background-color: #b1b1b1;
    }
    #footer {
        width: 100%;
        height: 100px;
        background-color: #979797;
    }
</style>

02. 구조 잡기

헤더 영역에 로고와 네비게이션 영역을 작업하겠습니다. 헤더 박스에 로고 박스와 네비게이션 박스를 넣는다고 생각하시면 됩니다. 헤더 태그 안에 h1 태그와 nav 태그를 설정하겠습니다. 

<header id="header">
    <h1></h1>
    <nav></nav>
</header>

헤더 영역을 확인하기 위해 width, height, background-color을 설정하겠습니다. 이렇게 하면 헤더 영역이 완성되었습니다. 간단하죠? 아직은 이해 못하고 있는 부분이 있어도 하나씩 따라하면서 천천히 이해하시면 됩니다. 

#header {
    width: 100%;
    display: flex;
}
#header h1 {
    width: 20%;
    height: 100px;
    background-color: #ffeded;
}
#header nav {
    width: 80%;
    height: 100px;
    background-color: #ffe0e0;
}

이번에는 이미지 영역을 작업해보겠습니다. 이미지 영역은 별다른게 없기 때문에 배경색만 변경하고, 컨텐츠 영역을 작업하겠습니다. 컨텐츠 영역안에는 3개의 박스가 들어가기 때문에 3개의 박스를 가로로 정렬하기 위해서 display: flex를 사용하고, width, height, background-color를 설정하겠습니다. 

<article id="slider">
</article>
<!-- //slider -->

<section id="contents">
    <div class="content1"></div>
    <div class="content2"></div>
    <div class="content3"></div>
</section>

#slider {
    width: 100%;
    height: 300px;
    background-color: #ffd7d7;
}
#contents {
    width: 100%;
    display: flex;
}
.content1 {
    width: 33.3333%;
    height: 200px;
    background-color: #ffcccc;
}
.content2 {
    width: 33.3333%;
    height: 200px;
    background-color: #fcbebe;
}

푸터 영역 역시 3개의 박스로 구성되어 있기 때문에 3개의 박스를 만들고 가로로 정렬하겠습니다. 영역이 제대로 잡혔는지 확인하기 위해서 width, height, background-color를 설정하겠습니다. 

<footer id="footer">
    <div class="footer1"></div>
    <div class="footer2"></div>
    <div class="footer3"></div>
</footer>
<!-- //footer -->

#footer {
    width: 100%;
    display: flex;
}
.footer1 {
    width: 20%;
    height: 100px;
    background-color: #ffa4a4;
}
.footer2 {
    width: 60%;
    height: 100px;
    background-color: #fc9292;
}
.footer3 {
    width: 20%;
    height: 100px;
    background-color: #ff8686;
}
