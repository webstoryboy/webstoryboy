소개

안녕하세요! 이번에는 웹디자인 기능사 레이아웃 두번째 시간입니다. A-1유형은 A-2유형은 동일합니다. 레이아웃을 처음 해보시는 분들은 다시 한번 따라해보는게 좋구요! 어는 정도 알고 있는 분들은 A-3유형으로 가셔도 됩니다. A유형을 충분히 이해한 분들은 B유형으로 가셔도 됩니다. 듣는 분들의 수준이 다 다르니 본인이 선택하셔서 참고했으면 좋겠습니다.

01. 기본 레이아웃

VSCODE를 실행하고 webd2023폴더를 만들고 webd02.html파일을 만들겠습니다.
!를 치고 tab버튼을 누르면 다음과 같이 작성하겠습니다. lang는 ko로 변경하고 title도 변경해주었습니다. 기본 레이아웃과 영역 설정을 작업하겠습니다. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>웹디자인 기능사 레이아웃 A-2</title>
    <style>
        * {
            margin: 0;
            padding: 0;
        }
        #wrap {
            width: 1200px;
            height: 700px;
            background-color: #e0e0e0;
            margin: 0 auto;
        }
        #header {
            width: 100%;
            height: 100px;
            background-color: #d1d1d1;
        }
        #slider {
            width: 100%;
            height: 300px;
            background-color: #bdbdbd;
        }
        #contents {
            width: 100%;
            height: 200px;
            background-color: #b1b1b1;
        }
        #footer {
            width: 100%;
            height: 100px;
            background-color: #979797;
        }
    </style>
</head>
<body>
    <div id="wrap">
        <header id="header"></header>
        <article id="slider"></article>
        <section id="contents"></section>
        <footer id="footer"></footer>
    </div>
</body>
</html>

02. 구조 잡기

헤더 영역에는 두개의 영역으로 구성되어 있습니다. 로고와 네비게이션으로 나누어 작업하겠습니다. 

<header id="header">
    <h1></h1>
    <nav></nav>
</header>
<!-- //header -->

#header {
    width: 100%;
    display: flex;
}
#header h1 {
    width: 20%;
    height: 100px;
    background-color: #daffea;
}
#header nav {
    width: 80%;
    height: 100px;
    background-color: #ccf6de;
}

슬라이드 영역은 작업할께 없으니 백그라운 색만 변경하겠습니다. 콘텐츠 영역은 3개의 박스로 구성되기 때문에 3개의 박스를 만들면 블록구조이기 때문에 가로 정렬이 되지 않고 세로 정렬로 됩니다. 구조가 깨진것처럼 보이지만ㅡ 3개의 부모 박스한테 display: flex를 설정하여 가로로 정렬 하겠습니다. 

<article id="slider">
</article>
<!-- //slider -->

<section id="contents">
    <article class="content1"></article>
    <article class="content2"></article>
    <article class="content3"></article>
</section>
<!-- //contents -->

#slider {
    width: 100%;
    height: 300px;
    background-color: #b0ffd2;
}
#contents {
    width: 100%;
    display: flex;
}
.content1 {
    width: 33.3333%;
    height: 200px;
    background-color: #8fffbf;
}
.content2 {
    width: 33.3333%;
    height: 200px;
    background-color: #85edb2;
}
.content3 {
    width: 33.3333%;
    height: 200px;
    background-color: #78e0a5;
}

푸터 영역도 3개의 자식 박스들로 구성되어 있습니다. 각각의 박스를 만들고 class 이름을 만들고, width, height, background-color를 설정합니다. 

<footer id="footer">
    <div class="footer1"></div>
    <div class="footer2"></div>
    <div class="footer3"></div>
</footer>
<!-- //footer -->

.footer1 {
    width: 20%;
    height: 100px;
    background-color: #69c591;
}
.footer2 {
    width: 60%;
    height: 100px;
    background-color: #5baf80;
}
.footer3 {
    width: 20%;
    height: 100px;
    background-color: #4c9c6f;
}
