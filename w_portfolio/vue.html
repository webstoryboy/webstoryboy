    Vue Api Site

    • Period:
      22/09 ~ 22/10
    • Framework:
      Vue
    • Libraries:
      Swiper
    • API:
      Youtube, TMDB, Unsplash

  • Vue Api Site는 React프레임워크를 이용해 여러 API소스를 활용하여 만든 프로젝트입니다. Vue 프레임워크의 장점을 활용해 페이지간 이동이 빠르고, API를 이용해 다채로운 컨텐츠를 만들었습니다.

  • Youtube, TMDB, Unsplash 등 다양한 API소스를 fetch 방식으로 불러서 게시물 검색, 슬라이더, 바로가기 등의 기능을 만들었습니다.

  • Swiper 라이브러리를 이용해 간단한 슬라이더를 구현했습니다.
